# DevConnectMorocco


Plateforme web communautaire dédiée à l'écosystème IT marocain.

## Stack Technique
- **Backend** : Laravel 11 (PHP 8.3) + Sanctum
- **Frontend** : React.js 18 + Vite + Tailwind CSS
- **Base de données** : MongoDB 7.x
- **Cache** : Redis
- **IA** : OpenAI API

## Structure
- `/backend` → API Laravel
- `/frontend` → App React

## Lancer le projet

### Backend
cd backend && php artisan serve

### Frontend
cd frontend && npm run dev
